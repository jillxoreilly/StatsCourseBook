We noted previously that our climate dataset is not suitable for a â€˜normalâ€™ Pearsonâ€™s correlation because it has one of the three freatures that violate the assumptions of Pearsonâ€™s correlation:

  • Non-Straight-Line relationship between x and y
  • Heteroscedasticity
  • Outliers

In this case the problem was heteroscedasticity

Spearmanâ€™s Rank Correlation#

We therefore use a rank-based form of correlation called Spearmanâ€™s rank correlation coefficient \(r_s\), which does not rely on the same assumptions as Pearsonâ€™s correlation.

\(r_s\) can be calculated using the same built-in function pandas.df.corr():

CO2vGDP.corr(method='spearman')
CO2 GDP population
CO2 1.000000 0.913841 -0.091860
GDP 0.913841 1.000000 -0.119114
population -0.091860 -0.119114 1.000000

The â€˜methodsâ€™ available are:

  • Pearson ('normal' correlation),
  • Spearman (rank based) and
  • Kendal (categorical; rarely used).

The default (the method used if no method is specified) is Pearson - we can see this by comparing the results with method specified as Pearson and no method specified:

CO2vGDP.corr(method='pearson')

The equation#

Letâ€™s check our understanding by calculating Spearmanâ€™s correlation coefficient oursleves.

The equation is:

\[ r_s = 6 \sum{\frac{d^2}{n(n^2 - 2)}} \]

â€¦ where d is the difference in ranks between paired datapoints.

Letâ€™s walk that through.

First, we create new columns containing the rank of each datapoint -

  • the lowest GDP will have rank 1 and the highest will have rank 165
  • the lowest CO2 emissions will have rank 1 and the highest will have rank 165
  • the GDP aand CO2 ranks for each country need not match, but if GDP and CO2 are correlated,
  • high-ranked countries for GDP should also be high-ranked for CO2 
CO2vGDP['CO2_rank'] = CO2vGDP['CO2'].rank()
CO2vGDP['GDP_rank'] = CO2vGDP['GDP'].rank()

# We can see these most clearly if we sort the dataframe before displaying
display(CO2vGDP.sort_values(by='CO2'))

To continue the process of applying the equation, we make a new column containing the difference of ranks:

CO2vGDP['d'] = CO2vGDP['CO2_rank']-CO2vGDP['GDP_rank']

â€¦ and apply the formula:

n = len(CO2vGDP)

r_s = 1 - 6*sum(CO2vGDP['d']**2)/(n*(n**2 - 2))
print('r = ' + str(r_s))

r = 0.913837563824707

Ta-daa! This should match the value from the built in function.

Spearman = Pearson on Ranks#

The equation for Spearmanâ€™s \(r_s\) looks pretty weird, doesnâ€™t it? What is that 6 doing there?!

I canâ€™t tell you the derivation (although I believe I did once know it), but I can tell you a fun thing.

Spearmanâ€™s \(r_s\) is exactly the same value as Pearsonâ€™s \(r\) claculated on the ranks.

Letâ€™s try it!
